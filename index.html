Common Questions

Straight
Answers

Is this therapy?

No. This is not therapy, and Steve is not a therapist. This is strategic conversation — direct, action-oriented work on where you are and where you’re going. If you need clinical support, please seek it. This is something different.

What if I’m not sure what I need?

That uncertainty is exactly what the session is designed to cut through. You don’t need to arrive with a clear agenda. Show up honest. The rest happens in real time.

How is one session enough?

One clear move is better than twelve sessions of circling. The goal isn’t dependency — it’s clarity. You’ll leave with something specific to do. What you do with it is up to you.

Can I book more sessions?

Yes. If one session opens something that needs continued work, Steve is available. But there’s no package, no pressure, and no manufactured timeline. You book when you need it.
